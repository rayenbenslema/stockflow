# StockFlow Design System

This document expands the UI/UX rules defined in `../../CONSTITUTION.md`.

## Rule

The locked Stripe-inspired design system in `CONSTITUTION.md` is mandatory.

## Priorities

- French-first UI
- Premium fintech feeling
- Clean spacing
- Mobile-first responsiveness
- Touch-friendly POS
- Fast tables
- Clear forms
- Accessible contrast

## Forbidden

- cluttered ERP layouts
- tiny buttons
- arbitrary spacing
- inconsistent colors
- non-responsive tables
