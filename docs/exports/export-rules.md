# Export Rules

This document expands the export rules defined in `../../CONSTITUTION.md`.

## Mandatory Export Formats

- Excel
- CSV
- PDF

## Exportable Entities

- invoices
- products
- inventory
- analytics
- clients
- suppliers
- payments
- TVA reports

## Excel Rules

Exports must:
- be readable
- use clear headers
- preserve totals
- include dates
- include business identity
- support accountant usage
