# Export Rules

This document expands the export rules defined in `../../CONSTITUTION.md` (§35-36). It does NOT override the constitution.

---

## Mandatory Export Formats

- Excel (.xlsx)
- CSV (.csv)
- PDF (.pdf)

## Exportable Entities

| Entity | Formats |
|---|---|
| Invoices | Excel, CSV, PDF |
| Products | Excel, CSV |
| Inventory | Excel, CSV |
| Analytics | Excel, CSV |
| Clients | Excel, CSV |
| Suppliers | Excel, CSV |
| Payments | Excel, CSV |
| TVA Reports | Excel, CSV, PDF |

## Excel Requirements

Exports MUST:

- be readable with clear grouped headers
- support filters on all columns
- preserve formatting and totals
- include dates and business identity
- support accountant usage (export comptable)
